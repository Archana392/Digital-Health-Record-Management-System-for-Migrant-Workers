# User & Module Identification

## Users

### Admin
- Manage Migrant Workers
- Manage Doctors
- Monitor System
- Generate Reports

### Doctor
- View Worker Records
- Add Medical Notes
- Update Diagnosis
- Manage Appointments

### Migrant Worker
- Register Account
- Login
- View Health Records
- Book Appointments
- Update Profile

---

## Modules

### Worker Registration Module
Stores worker personal information.

### Authentication Module
Handles login and access control.

### Health Record Module
Maintains medical history and records.

### Appointment Module
Manages doctor appointments.

### Doctor Management Module
Handles doctor information.

### Dashboard Module
Displays system information and statistics.
